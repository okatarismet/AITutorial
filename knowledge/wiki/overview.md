# What is AI Native Engineering?

> **Status: living stub.** A working definition, to be refined as sources are ingested.

**AI Native Engineering** is the discipline of designing, building, and operating software where large language models (and related AI systems) are a primary building block — not a bolt-on feature. An AI-native engineer treats the model as a core component of the architecture and is fluent in the skills that make that reliable: prompting, [[context-engineering|context engineering]], tool use, agents, retrieval, evaluation, and the production realities of cost, latency, and safety.

This wiki is the knowledge base for an education platform teaching that discipline. It is built and maintained using the [[../llm_wiki|LLM Wiki pattern]]: sources go into `raw/`, and Claude compiles them into the interlinked pages here.
